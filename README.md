## üèÅ Getting Started

To run this app locally:

1. Clone the repo  
   `git clone https://github.com/Milakrei55/CFAngularMyFlix.git`
2. Navigate to the project folder  
   `cd CFAngularMyFlix`
3. Install dependencies  
   `npm install`
4. Run the app  
   `ng serve`

Visit `http://localhost:4200/` in your browser to explore the app.
